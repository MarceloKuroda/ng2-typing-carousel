# ng2-typing-carousel
Simple typing carousel directive for angular2.
Thanks: http://codepen.io/dissimulate/pen/BvpGr
## Installation

To install this library, run:

```bash
$ npm install ng2-typing-carousel --save
```

## Development

To generate all `*.js`, `*.js.map` and `*.d.ts` files:

```bash
$ npm run tsc
```
